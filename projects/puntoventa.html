Punto de Venta

  • Github URL: Project Link

This project is created using C# and .Net Framework as backend. I have used Microsoft SQL Server as database and MVC pattern for a correct and efficient organization of the code.

The software consist of a POS(Point of Sale) system, for a School Project, where the user have a CRUD system for the products and employees. The user can sell the products and generate a report of the sales.
